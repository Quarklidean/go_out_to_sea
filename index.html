企业出海：
基于FDI的动机、路径与绩效分析

深度解析中国企业全球化战略的多维度框架与国际比较研究

核心洞察

企业出海动机从传统的市场和效率驱动，转向技术、品牌与战略资产寻求的多元化格局

全球视野

中国企业以供应链和数字技术优势，在南北方国家布局中展现独特竞争力

引言

企业出海，即通过对外直接投资（FDI）在全球范围内配置资源、拓展市场，已成为企业寻求增长、提升竞争力的核心战略。本文基于FDI的理论框架，系统梳理了企业出海的核心动机、目标国选择、路径决策及绩效评价体系。

研究背景

研究发现，企业出海动机已从传统的市场和效率驱动，转向技术、品牌与战略资产寻求的多元化格局。在目标国选择上，呈现出对发达国家（北方国家）的技术与市场，以及发展中国家（南方国家）的成本与市场潜力并重的趋势。

核心发现

通过对中国企业与欧美、日韩、印度企业的国际比较，揭示了中国企业出海的独特性：以强大的供应链和制造能力为基础，以领先的数字技术为赋能，并依托超大规模的国内市场作为战略支撑。

文献综述：企业FDI的理论框架与演进

企业出海的核心动机分析

企业对外直接投资（FDI）的动机是复杂且多维度的，受到全球经济格局、技术周期、地缘政治以及企业自身发展阶段等多重因素的影响。近年来，中国企业出海的动机呈现出显著的结构性变迁，从早期的资源寻求和成本驱动，转向更为复杂的市场、技术和战略驱动。

市场寻求型动机

根据fDi Markets的数据，在2018年至2023年间，"接近市场与客户"以40%的占比高居榜首，成为最核心的驱动力[1]。这标志着中国企业出海已经从单纯的产品输出，深化到服务本地市场、建立本土化运营体系的新阶段。

"无锡日联科技股份有限公司在马来西亚设立首个海外工厂，正是看中了东南亚市场对中国新技术、新装备的青睐。"

资源与效率寻求型动机

"获取熟练劳动力"占比22%，显示出中国企业对高端人力资源的渴求。随着国内产业升级和技术进步，企业对高技能、创新型人才的需求日益增长。

"中国企业'抱团'出海，利用中国强大的供应链和配套制造能力，在海外建立产业集群。"

目标国选择：南北方国家的区位因素

企业在进行对外直接投资时，目标国的选择是决定投资成败的关键一步。这一决策过程受到目标国区位优势的深刻影响，这些优势通常可以概括为两大类：一是以发达国家（北方国家）为代表的技术、市场和制度优势；二是以发展中国家（南方国家）为代表的低成本、资源和市场潜力优势。

目标北方国家（发达国家）

  • 技术先进：世界领先的研发能力、创新生态和高端人才储备
  • 市场成熟：规模庞大、消费能力强且需求多元化的消费市场
  • 制度完善：健全的法律体系、透明的监管环境和稳定的政治经济秩序

目标南方国家（发展中国家）

  • 成本优势：较低的劳动力成本、土地成本和原材料成本
  • 资源丰富：矿产、能源、农产品等自然资源丰富
  • 市场潜力：经济快速增长，庞大的人口基数和不断壮大的中产阶级

出海路径选择：绿地投资与跨国并购

企业在确定了出海动机和目标国之后，接下来面临的关键决策是选择何种路径进入海外市场。对外直接投资（FDI）的主要路径可以分为两大类：绿地投资（Greenfield Investment）和跨国并购（Cross-border M&A）。

绿地投资

核心优势
  • • 完全控制权，可定制化布局
  • • 获得当地政府支持
  • • 长期品牌建设
适用情境

资本实力雄厚、产品复购率高、对本地响应速度要求强的龙头企业

跨国并购

核心优势
  • • 快速获取现有资产
  • • 缩短市场进入时间
  • • 获得技术和渠道
适用情境

希望快速获取关键技术、填补市场空白或应对激烈竞争的企业

路径选择关键因素

企业能力：生产率、研发密度、营销网络发达程度
行业特征：竞争强度、技术更新速度、资本密集度
目标国环境：制度完善度、市场成熟度、文化差异

出海绩效评价：多维度综合评估

对企业出海绩效的评价，不能仅仅局限于财务指标，而应建立一个涵盖财务、市场、创新和战略等多个维度的综合评价体系。这是因为企业出海的动机是多元化的，其成功与否也应从多个角度进行衡量。

财务绩效

最直观、最核心的指标，包括投资回报率（ROI）、净资产收益率（ROE）、利润率等。

"近60%的受访中国企业在2024年实现盈利，超过半数的企业利润率相比2023年有所提升"

[2]

市场绩效

衡量企业在目标市场的地位和影响力，包括市场份额增长、品牌知名度提升等。

比亚迪在东南亚投资建厂，目标不仅是销售产品，更是要在当地新能源汽车市场占据领先地位。

创新绩效

衡量企业在技术创新和研发能力方面的进步，包括核心技术获取、专利数量等。

通过收购德国、美国的科技公司，中国企业获得了先进技术，更重要的是吸收了研发管理经验。

战略绩效

从宏观、长远角度评价对企业整体战略目标的贡献，包括全球价值链地位提升等。

通过国际化经营，企业实现了全球研发、全球生产、全球销售的战略布局。

行业结构转变

制造业升级
  • • 从采矿业为主转向制造业为主
  • • 集中于金属、电子元器件、汽车

民营企业成为出海主力

在中国企业出海的浪潮中，一个尤为显著的特征是民营企业已经取代国有企业，成为出海的主力军。与早期以大型国企为主导的资源型和基础设施投资不同，近年来的出海进程主要表现为以民营企业为代表的全球化进程。

民营企业出海特征
灵活性优势：更加灵活、市场嗅觉更敏锐，敢于在陌生海外市场探索尝试
供应链优势：背靠中国强大的供应链和配套制造能力，"抱团"出海
数字化渠道：积极利用数字化平台和跨境电商等新兴渠道
品牌形象：从低成本制造向创新、品牌和服务形象转变

中国企业出海的独特优势

中国企业在全球竞争中展现出了一系列独特的优势，这些优势根植于中国特殊的经济体制、市场规模和产业生态，构成了其成功出海的重要基石。这些优势并非单一因素，而是一个相互关联、相互强化的系统。

制造业优势

经过几十年的发展，中国已经形成了全球最完整、最复杂的工业门类体系，从原材料、零部件到最终产品，几乎所有环节都能在国内找到高效、低成本的供应商。

"供应链优势 = 强大的生产制造能力 + 领先的数字基础设施 + 不断加密的国际物流网络"

数字化优势

中国拥有全球领先的数字基础设施，在电子商务、社交媒体、数字内容等领域积累了丰富的运营经验和创新的商业模式。

字节跳动（TikTok）成功将直播带货、社交电商、算法推荐等模式推向全球市场。

市场优势

超大规模国内市场为企业出海提供了坚实的大后方和独特的战略优势，包括充足的"练兵场"和强大的议价能力。

"先国内，后国际"的发展路径，为参与全球竞争奠定了坚实基础。

国际比较：中国企业与欧美日韩印企业的出海战略

在全球化的浪潮中，不同国家和地区的企业在出海战略上展现出各自鲜明的特点。这些差异根植于其母国的经济发展阶段、产业结构、文化背景以及政府政策。通过将中国企业与欧美、日韩、印度等代表性经济体的企业进行比较，可以更清晰地揭示中国企业出海的独特性。

出海动机的比较

国家/地区 核心出海动机 典型特征与案例
中国企业
市场与效率驱动为主，技术与战略资产寻求为辅 早期以开拓市场、降低成本为主，近年来日益重视通过海外投资获取先进技术、品牌和管理经验。例如，对东南亚的投资多为市场与效率驱动，对欧美的投资则多为技术与品牌驱动。
欧美企业
技术与品牌输出为主 凭借在全球价值链顶端的技术、品牌和管理优势，通过FDI在全球范围内推广高附加值产品和服务。例如，德国的汽车和机械、美国的互联网和软件、法国的奢侈品等。
日韩企业
资源与市场并重 历史上因资源匮乏，出海带有强烈的资源寻求色彩。同时，高度重视开拓海外市场，特别是在汽车和电子等制造业领域，通过全球化生产和销售网络，成为全球领先品牌。
印度企业
服务与技术输出为主 凭借在信息技术（IT）、软件外包和业务流程外包（BPO）领域的优势，为全球客户提供服务。近年来，制药、汽车等行业的企业也开始积极进行海外扩张。

中国企业：多元化动机驱动

中国企业出海的核心动机，在初期主要表现为市场和效率驱动。随着中国经济转型升级，技术与战略资产寻求动机愈发凸显。特别是在对发达国家的投资中，获取先进技术、知名品牌和全球营销网络成为重要目标。

"制度与政策驱动也是中国出海的一个显著特征，例如响应'一带一路'倡议，或在墨西哥等地投资以规避美国关税壁垒。"

欧美企业：技术与品牌输出

欧美企业作为全球跨国公司的先驱，其出海战略的核心是技术和品牌输出。凭借其在科技、研发、品牌管理和全球营销方面的长期积累，致力于在全球范围内推广其高附加值的产品和服务。

近年来，随着全球对可持续发展的关注，欧美企业在绿色技术、清洁能源等领域的投资也日益增多。

以亚洲邻国和北美市场为主 亚洲邻国因其地理、文化和经济联系成为首选投资地。北美作为全球最大的消费市场，是其必争之地。

地缘政治与区域经济合作的影响

地缘政治和区域经济合作对企业目标国选择的影响日益显著。一方面，地缘政治冲突和贸易摩擦会促使企业调整其全球供应链布局，以规避风险。另一方面，区域经济一体化和自由贸易协定的签署，为企业创造了更加便利的投资和贸易环境。

挑战：中美贸易摩擦导致"中国+1"战略，企业将产能转移到越南、墨西哥等国
机遇："一带一路"倡议、非洲大陆自由贸易区等区域合作创造新机遇

中国企业：灵活选择

中国企业的出海路径呈现出绿地投资与跨国并购并举的特点。联想收购IBM PC业务、吉利收购沃尔沃等都是中国企业通过跨国并购实现跨越式发展的典型案例。

欧美企业：并购主导

欧美企业由于其强大的资本实力和丰富的并购经验，其海外投资更多地采用跨国并购的方式，以快速进入新市场、消灭竞争对手、获取战略性资产。

日韩企业：绿地投资

日本和韩国企业的海外投资则更多地采用绿地投资的方式，希望将其独特的管理模式、企业文化和生产体系完整地复制到海外。

案例分析：不同产业类型的出海实践

传统制造业出海案例

传统制造业的出海，通常表现为寻求更低的生产成本、更广阔的市场以及更稳定的资源供应。中国企业在该领域的出海，往往伴随着技术升级和产业链转移的双重目标。

中电建在东南亚的基础设施投资

中国电力建设集团有限公司在东南亚的投资展示了大型基建企业通过FDI进入南方发展中国家市场的典型案例。代表性项目包括老挝南欧江流域梯级水电站项目和印尼雅万高铁项目。

核心动机
  • • 市场寻求：东南亚基础设施建设需求旺盛
  • • 技术输出：将中国在大型水电、高铁领域的技术输出
  • • 品牌提升：提升中国基建品牌的国际影响力

徐工机械在南美、非洲的工程机械投资

徐工集团通过绿地投资与建立海外销售网络并举的方式，成功进入南美和非洲等关键市场。在巴西建立了南美最大的制造基地，在非洲建立了完善的销售和服务网络。

核心动机
  • • 市场寻求：南美、非洲基础设施建设需求巨大
  • • 效率寻求：降低物流成本，规避关税壁垒
  • • 本土化运营：深耕当地市场，建立可持续竞争优势

国际比较：日本汽车企业在东南亚的投资布局

日本汽车企业在东南亚的投资布局是传统制造业全球化扩张的经典案例。早在20世纪70年代，丰田、本田等便开始在泰国、印度尼西亚等东南亚国家进行大规模投资。

生态出海模式：不仅在当地生产整车，还积极培育本土的零部件供应商，形成了紧密的产业协同网络
深度整合：丰田在泰国设厂时，会带动其核心零部件供应商如电装、爱信等一同出海，在当地建立配套工厂

新兴制造业出海案例

新兴制造业的出海，往往伴随着技术、资本和品牌的全球竞争。中国企业在新能源汽车、电池等领域，正从过去的"追赶者"转变为"并跑者"甚至"领跑者"，其出海战略也呈现出技术寻求与市场开拓并重的特点。

宁德时代在德国、匈牙利的新能源电池投资

宁德时代作为动力电池制造商，在欧洲汽车工业的核心地带建立大型生产基地。在德国投资建设电池工厂，在匈牙利德布勒森投资73.4亿欧元建设欧洲第二座工厂。

战略考量
  • • 市场驱动：直接服务于宝马、奔驰等德国主流车企
  • • 技术吸收：学习欧洲在工业4.0、智能制造方面经验
  • • 战略卡位：地处欧洲中心，辐射整个欧洲市场

比亚迪在东南亚的新能源汽车投资

比亚迪凭借其领先的电池技术、电控技术和垂直整合的产业链优势，积极布局海外市场。在泰国罗勇府投资建设的电动汽车工厂是其首个海外乘用车生产基地。

战略考量
  • • 市场寻求：东南亚人口众多，经济增长迅速，新能源汽车需求旺盛
  • • 效率提升：降低生产和运输成本，规避贸易壁垒
  • • 区域辐射：作为向澳大利亚、新西兰等地区出口的跳板

国际比较：德国汽车企业在中国的投资布局

德国汽车企业在中国的投资布局是国际新兴制造业合作的典范。自20世纪80年代以来，大众、宝马、奔驰等便进入中国，与中国本土企业成立合资企业。

战略演进：从"技术换市场"到"在中国，为中国"，再到"在中国，为世界"
新投资：大众投资25亿欧元拓展合肥生产及创新中心，宝马投资300亿元人民币升级沈阳工厂
本土化：深度融入中国市场，与中国合作伙伴共同成长，根据中国市场变化及时调整战略

传统服务业出海案例

传统服务业的出海，如餐饮、酒店等，其核心挑战在于如何在保持品牌特色的同时，实现深度的本地化，以适应不同国家和地区的文化、口味和消费习惯。中国餐饮企业的出海，正从早期的"移民餐饮"向"品牌餐饮"转型。

蜜雪冰城在东南亚的连锁扩张

蜜雪冰城作为中国新式茶饮的代表，将东南亚作为出海的第一站。主要采用加盟模式进行快速扩张，推出相对优惠的加盟政策，以"蚂蚁雄兵"式的布局迅速渗透市场。

成功要素
  • • 文化相似性：地理位置毗邻，茶文化相似
  • • 市场契合：年轻人口多，消费水平与品牌定位契合
  • • 供应链优势：自建供应链体系，低成本高品质

华住酒店在海外的酒店业投资

华住酒店集团主要通过战略投资和品牌合作的方式进行全球化扩张。投资德意志酒店集团（DH），实现"双向赋能"的模式，将华住品牌推向欧洲市场。

战略特点
  • • 轻资产模式：避免重资产自建酒店的高风险
  • • 双向赋能：引入国际品牌，同时输出中国品牌
  • • 服务提升：学习国际先进酒店管理经验

国际比较：美国快餐连锁企业在全球的扩张

美国快餐连锁企业在全球的扩张是传统服务业出海的经典案例。以麦当劳、肯德基、星巴克等为代表，其核心是将其标准化的产品、服务和品牌形象复制到全球各地。

标准化运营：在食品制作工艺、口味、装修风格、服务流程等方面实现全球统一
特许经营：通过特许经营模式快速扩张，降低市场进入风险和成本
本土化策略：在坚持核心产品标准化的同时，根据当地口味推出本土化产品
成功案例：肯德基在中国推出豆浆、油条等中式早餐，深受消费者喜爱

新兴服务业出海案例

新兴服务业的出海，如互联网、金融科技、云计算等，是全球化和数字化时代的新现象。中国企业在这一领域展现出强大的竞争力，其出海战略往往以技术和模式创新为核心，通过数字平台连接全球用户。

字节跳动（TikTok）在欧美的数字服务投资

TikTok的出海路径主要是通过收购和本土化运营相结合的方式。2017年，字节跳动斥资10亿美元收购美国短视频应用Musical.ly，并将其与TikTok合并，快速切入欧美市场。

成功要素
  • • 算法技术：强大的算法推荐技术，提供个性化内容
  • • 本土化运营：在各国设立本地化运营团队
  • • 模式复制：将中国市场验证成功的运营模式引入欧美
挑战

面临日益严峻的监管压力和地缘政治风险，如美国国家安全审查、欧盟《数字服务法案》监管等

菜鸟网络在东南亚的物流网络建设

菜鸟网络通过投资、合作和自建相结合的方式，在东南亚构建高效、智能的物流网络。在马来西亚吉隆坡建立首个海外eWTP超级物流枢纽。

战略特点
  • • 基础设施先行：通过构建物流基础设施赋能区域发展
  • • 生态构建：服务于阿里巴巴在东南亚的电商业务
  • • 区域赋能：赋能当地中小企业，促进贸易便利化

国际比较：美国互联网公司在中国的投资布局

美国互联网公司在中国的投资布局是国际新兴服务业合作与竞争的缩影。以谷歌、亚马逊、微软、苹果等为代表的企业自20世纪90年代末便纷纷进入中国。

挑战：严格的网络内容审查、激烈的市场竞争、本土企业快速崛起，导致许多美国公司在华业务遭遇挑战
地缘政治：随着中美科技竞争加剧，美国对华投资特别是高科技领域投资受到越来越严格的审查，反映了新兴服务业出海面临的地缘政治风险

"蚂蚁金服收购美国速汇金（MoneyGram）的交易就因国家安全顾虑而告吹，体现了新兴服务业出海面临的地缘政治风险。"
